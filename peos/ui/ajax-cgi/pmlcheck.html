PMLcheck Submit Form

This is a very minimal pmlcheck interface designed to demonstrate the round-trip process for submitting pml code to a server-side cgi program. It submits the PML code from the editor window to the server via jquery's ajax() method, then parses the pmlcheck output and inserts error messages into the editor display.

The editor is codemirror, which has textarea replacement built-in; in principle there is no reason why Ace or any other javascript-based editor could be used.

For this demo to work properly, you must install the codemirror distribution into the same directory where this page is found, in a subdirectory called "codemirror".

At present the icons for the error messages do not display correctly, even though I used the code from the demo almost verbatim.
