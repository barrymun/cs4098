Installation

Requirements:

•  Linux Workstations

•  Xerces 2.6.2

•  jdk 1.4.2

•  PEOS source code

Specifically, to define their own processes, PEOS users are required to use PML (Process Modeling Language) and its compiler, which are bundled as part of the overall PEOS code-base. Thus, if users want to define processes, they must not only have access to the PEOS source code (or at least selected executables), but also be able to run the PML compiler on a Linux/Unix platform.
