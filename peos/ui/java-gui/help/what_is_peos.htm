What is PEOS?

The Process Enactment Operation System (PEOS) incorporates the capability to translate a PML (Process Modeling Language)-defined process to a Linux-compatible format, instantiate it at the user's request, and run the process on the platform as dictated by the user.

PEOS is capable of running any sequential process (defined in and compiled from PML) end to end, and displaying to the user the pre-defined instructions – if applicable – that explain each action/task in the process. In addition, it implements the functionality to allow a PEOS user to suspend any running process at any point, start a new process (even another instance of the same process), and then resume the first process from the point of suspension.
